### 📖 Deskripsi Singkat
Ini adalah **Base Murni**. Project ini fokus menyediakan struktur folder dan inisialisasi awal agar kamu tidak perlu membangun semuanya dari nol. Dibuat sesederhana mungkin agar mudah dipahami bahkan bagi yang baru belajar.

### ⚠️ Informasi Penting
> [!IMPORTANT]
> Proyek ini adalah **kerangka kosong** (Tanpa Database & Tanpa User State).

* **Tanpa Database:** Belum ada integrasi database (seperti MongoDB, MySQL, dll).
* **Tanpa User State:** Belum ada sistem penyimpanan status user (*state* atau *session*).
* **Fleksibel:** Kamu bebas memasang database atau sistem login apapun sesuai kebutuhan proyekmu.

<hr />

### 📁 Struktur Folder

| Folder / File | Deskripsi |
| :--- | :--- |
| `commands/` | Tempat menyimpan logika perintah (handler, menu, callback). |
| `config/` | Pengaturan bot, tempat menaruh API Token kamu. |
| `lib/` | Library tambahan atau fungsi pembantu (utilitas). |
| `index.js` | Titik masuk utama (otak bot) untuk menjalankan aplikasi. |

<hr />

### 🛠️ Cara Penggunaan

1. **Ambil Source Code**
```bash
git clone [https://github.com/Zeallord/Base-Bot-Telegram.git](https://github.com/Zeallord/Base-Bot-Telegram.git)
cd Base-Bot-Telegram
````
2.  **Instalasi Library**

<!-- end list -->

```bash
npm install
```

3.  **Konfigurasi**
    Buka folder `config/` dan isi API Token bot kamu yang didapat dari @BotFather.

4.  **Jalankan Bot**

<!-- end list -->

```bash
npm start
```
